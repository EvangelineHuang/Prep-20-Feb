Reduce Method

The syntax of Array's reduce method is array.reduce(function(total, currentValue, currentIndex, arr), initialValue)

The total is the accumulator. The function is executed for each element of array, and current value is added to the accumulator. After executing all the elements, the total is added to the initialValue, which is optional. If there is not an initialValue, it starts at 0 automatically.

If we want to find the mean using reduce method, first we need to create an array=[11,4,41,50,7,19,8,18,2,6,43,6,20,43,2,23,12,24,31,29] the entire code is:
array.reduce(function(total,current){
return total+current;},0)

Then we use array's length property to get the number of elements in the array

Finally, we divided the value that returned by reduce function by the length and get the mean.
